--------- 파일 dictionary.txt의 내용 --------- 
apple의 뜻은 사과입니다 
banana의 뜻은 바나나입니다 
carrot의 뜻은 당근입니다 
dog의 뜻은 개입니다 
egg의 뜻은 달걀입니다 
fox의 뜻은 여우입니다 

사전이름을 입력하세요: C:\Users\user\Downloads\dictionary.txt
영어단어를 입력하세요: banana
banana의 뜻은 바나나입니다 


사전이름을 입력하세요: C:\Users\user\Downloads\dictionary.txt

알고리즘

프로그램 시작 
   먼저, 영한 사전을 준비 
   영어 단어를 입력 
   영한 사전에서 한 줄씩 읽고 
      해당 영어단어를 찾으면 뜻을 출력 
   영한 사전 닫기 
프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter16/DictionaryFile.java

	import java.io.*;

	import java.util.Scanner;

		 

	public class DictionaryFile

	{

		// 프로그램 시작 

1		public static void main( String[] args ) throws IOException {

			Scanner scan = new Scanner( System.in );

			// 먼저, 영한 사전을 준비 

			System.out.print( "사전이름을 입력하세요: " );

2			String filename = scan.next();

3			BufferedReader dictionary = new BufferedReader( new FileReader( new File( filename ) ) );

			 

			// 영어 단어를 입력 

			System.out.print( "영어단어를 입력하세요: " );

5			String word = scan.next();

		 

			// 영한 사전에서 한 줄씩 읽고 

6			for ( String meaning = ""; ( meaning = dictionary.readLine() ) != null; ) {

				// 해당 영어단어를 찾으면 뜻을 출력 

7				if ( meaning.contains( word ) ) {

					System.out.println( meaning );

8					break;

				}

			}

	 

			// 영한 사전 닫기 

9			dictionary.close();

			scan.close();

		// 프로그램 종료 

10		}

	}
