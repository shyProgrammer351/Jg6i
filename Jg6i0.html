16. 파일입출력

파일    


파일 입력 스트림
File, FileReader
read, close		     프로그램    


파일 출력 스트림
File, FileWriter
write, close		     파일

File   파일을 관리하는 클래스

FileReader   파일에서 데이터를 읽어오는 클래스

FileWriter   파일에 데이터를 쓰는 클래스

InputStreamReader   바이트 스트림을 문자 스트림으로 변환하여 데이터를 읽어오는 클래스

OutputStreamWriter   문자 스트림을 바이트 스트림으로 변환하여 데이터를 출력하는 클래스

BufferedReader   효율성을 고려하여 버퍼에 데이터를 모았다가 한꺼번에 읽어오는 클래스

BufferedWriter   효율성을 고려하여 버퍼에 데이터를 모았다가 한꺼번에 쓰는 클래스

readLine   데이터를 한 줄씩 읽어오는 메서드

read   읽기 메서드

write   쓰기 메서드

flush   버퍼에 남아 있는 데이터를 밀어내기

close   파일 닫기

IOException   입출력시 발생할 수 있는 예외를 처리하기 위한 클래스
