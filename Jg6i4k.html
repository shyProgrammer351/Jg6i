문제

아래 설명된 의미에 해당하는 용어를 쓰세요 
 파일을 관리하는 클래스  

 효율성을 고려하여 버퍼에 데이터를 모았다가 한꺼번에 읽어오는 클래스  

 파일에서 데이터를 읽어오는 클래스  

 데이터를 한 줄씩 읽어오는 메서드  

 리소스 닫기 메서드  

 파일 입출력시 발생할 수 있는 예외를 처리하기 위한 클래스  

 해당 메서드에서 예외가 발생하면 지정한 예외 클래스로 보내서 처리  

 객체가 없음  

 -2, -1, 0, 1, 2 등의 정수를 표현하는 기본자료형  

 정수 자료형의 2차원 배열  

 "apple", "banana" 등을 표현하는 문자열 클래스  

 배열의 길이  

 문자열에서 괄호안의 인덱스번호에 해당하는 문자 가져오는 메서드  

 초기식, 조건식, 증감식으로 구성되고, 조건식이 참일 때마다 해당 명령문을 실행하는 반복문  

 조건식이 참이면 해당 명령문을 실행하는 선택문  

 앞에 제시된 조건식이 모두 거짓이면 해당 명령문을 실행하는 선택문  

 실행흐름을 중단하고 해당 영역을 빠져나오는 명령문  

 논리연산자 AND는 왼쪽과 오른쪽 조건식이 둘 다 모두 참이면 참을 반환하고, 나머지는 거짓을 반환  

 비교연산자로 두 값이 다른지 비교하는 등가연산자  

 비교연산자로 왼쪽 값이 오른쪽 값보다 작거나 같으면 참을 반환하고 아니면 거짓을 반환
